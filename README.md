**SI-KOS (SISTEM INFORMASI KOS)**

SI-KOS adalah aplikasi website berbasis Google Apps Script yang dikembangkan sebagai project mandiri untuk membantu pengelolaan data kos secara digital, sederhana, dan efisien.

Aplikasi ini menggunakan Google Sheets sebagai database, sehingga tidak memerlukan setup server database konvensional.

🚀 Fitur Utama
1. Sistem login sederhana
2. Manajemen data penyewa
3. Manajemen data kamar
4. Status penyewa (aktif / keluar)
5. Hapus data penyewa (khusus yang sudah keluar)
6. Upload & preview gambar kamar

💰 Pembayaran & Keuangan
1. Pencatatan pembayaran penyewa
2. Cetak kwitansi pembayaran penyewa
3. Riwayat pembayaran

📊 Laporan & Analisis
1. Laporan pembayaran
2. Grafik pembayaran (visualisasi data)
3. Monitoring pemasukan kos

📋 Dashboard
1. Ringkasan data (penyewa, kamar, pembayaran)

🛠 Teknologi yang Digunakan
Frontend: HTML, CSS, JavaScript
Backend: Google Apps Script
Database: Google Sheets
Deployment: Web App (Google Apps Script)
Version Control: GitHub
Development Tool: CLASP
